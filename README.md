**LifeStore** — бу чакана савдо дўконлари ва кичик корхоналар учун мўлжалланган, юқори хавфсизлик ва кенг имкониятларга эга бўлган бошқарув тизими. Дастур сотувларни тезлаштириш, омбор қолдиқларини назорат қилиш ва молиявий ҳисоботларни юритиш учун қулай ва замонавий интерфейс тақдим этади.

---

## ✨ Асосий Имкониятлар

### 🧾 Сотув ва POS (Kassir)
*   **Тезкор қидирув**: Маҳсулотларни номи ёки штрих-коди бўйича сонияларда топиш.
*   **Штрих-код сканер**: Барча турдаги сканерлар (USB/Wireless) билан ишлаш имконияти.
*   **Чек чиқариш**: Термал принтерларга (80мм/58мм) чек чиқариш ва созлаш.
*   **Мижозлар базаси**: Қарздорлик, содиқлик тизими ва мижозлар тарихини юритиш.

### 📦 Омбор ва Товарлар
*   **Категориялар**: Маҳсулотларни чексиз миқдордаги гуруҳларга ажратиш.
*   **Штрих-код Генератор**: Маҳсулотларга автоматик штрих-код яратиш ва этикетка (50x30мм) чиқариш.
*   **Инвентаризация**: Қолдиқларни назорат қилиш ва кам қолган товарлар ҳақида хабарнома олиш.

### 💸 Харажатлар Бошқаруви [NEW]
*   **Дўкон чиқимлари**: Ижара, ойлик, коммунал тўловлар ва бошқа харажатларни қайд этиш.
*   **Соф Фойда**: Савдо фойдасидан харажатларни айириб, ҳақиқий соф фойдани ҳисоблаш.

### 📊 Ҳисоботлар ва Статистика
*   **Дашборд**: Кунлик, ҳафталик ва ойлик савдо кўрсаткичларининг график кўриниши.
*   **Excel Export**: Барча ҳисоботларни Excel форматда юклаб олиш.
*   **Кириш тарихи**: Тизимга ким, қачон ва қайси қурилмадан кирганини кузатиш (Audit Log).

## 🛡️ Хавфсизлик ва Тизим

*   **Bruteforce Protection**: 5 марта хато пароль киритилса, IP-манзилни 15 дақиқага блоклаш.
*   **Auto-Logout**: 30 дақиқалик фаолиятсизликдан кейин сессияни автоматик ёпиш.
*   **Роллар тизими**: `Admin` ва `Cashier` роллари орқали кириш ҳуқуқларини чеклаш.

## 🤖 Автоматизация (Telegram)

*   **Telegram Bot Integration**: Янги сотувлар, харажатлар ва кам қолган товарлар ҳақида Telegram-га хабар юбориш.
*   **Авто-бэкап**: Маълумолтар базасини ҳар куни автоматик равишда белгиланган Telegram профилга юбориш.

## 🚀 Ўрнатиш ва Созлаш

### Талаблар:
*   **Сервер**: XAMPP, Laragon ёки OSPanel.
*   **PHP**: 7.4 ёки ундан юқори.
*   **MySQL**: 5.7 ёки ундан юқори.

### Қадамлар:
1.  Лойиҳани сервернинг `www` ёки `domains` папкасига юкланг.
2.  `init.sql` файлини MySQL базага импорт қилинг.
3.  `config.php` файлида база созламаларини ўзгартиринг:
    ```php
    $host = 'localhost';
    $user = 'root';
    $pass = 'your_password';
    $db   = 'lifestore';
    ```
4.  Тизимга кириш:
    - **URL**: `http://localhost/lifestore`
    - **Login**: `admin`
    - **Password**: `12345`
