# 🛒 LifeStore - Замонавий Савдо ва Омбор Тизими

**LifeStore** — бу чакана савдо дўконлари учун мўлжалланган, юқори хавфсизлик ва кенг имкониятларга эга бўлган бошқарув тизими. Дастур сотувларни тезлаштириш, омбор қолдиқларини назорат қилиш ва молиявий ҳисоботларни юритиш учун қулай интерфейс тақдим этади.

---

## ✨ Асосий Имкониятлар

### 🧾 Сотув ва POS (Kassir)
- **Тезкор қидирув**: Маҳсулотларни номи ёки штрих-коди бўйича сонияларда топиш.
- **Штрих-код сканер**: Барча турдаги сканерлар билан ишлаш имконияти.
- **Чек чиқариш**: Термал принтерларга (IP ёки Браузер орқали) чек чиқариш.
- **Мижозлар базаси**: Қарздорлик ва мижозлар тарихини юритиш.

### 📦 Омбор ва Товарлар
- **Категориялар**: Маҳсулотларни гуруҳларга ажратиш.
- **Штрих-код Генератор**: Маҳсулотларга автоматик штрих-код яратиш ва этикетка (50x30мм) чиқариш.
- **Инвентаризация**: Қолдиқларни назорат қилиш ва кам қолган товарлар ҳақида хабарнома олиш.

### 📊 Ҳисоботлар ва Статистика
- **Дашборд**: Кунлик, ҳафталик ва ойлик савдо кўрсаткичлари.
- **Фойда таҳлили**: Сотилиш ва таннарх орасидаги соф фойдани ҳисоблаш.
- **Кириш тарихи**: Тизимга ким, қачон ва қайси қурилмадан кирганини кузатиш.

### 🛡️ Хавфсизлик
- **Bruteforce Protection**: 5 марта хато пароль киритилса, IP-манзилни 15 дақиқага блоклаш.
- **Auto-Logout**: 30 дақиқалик фаолиятсизликдан кейин сессияни ёпиш.
- **Audit Log**: Барча муҳим амалларни ёзиб бориш.

### 🤖 Автоматизация
- **Telegram Bot**: Янги сотувлар ва кам қолган товарлар ҳақида Тelegram-га хабар юбориш.
- **Авто-бэкап**: Маълумотлар базасини ҳар куни автоматик равишда Telegram-га юбориш.

---

## 🚀 Ўрнатиш ва Созлаш

### Талаблар:
- **PHP**: 7.4 ёки ундан юқори.
- **MySQL/MariaDB**: 5.7 ёки ундан юқори (OSPanel тавсия этилади).
- **Браузер**: Google Chrome ёки Microsoft Edge.

### Қадамлар:
1.  Лойиҳани серверга юкланг (`C:/OSPanel/domains/lifestore`).
2.  `lifestore` номли маълумотлар базасини яратинг.
3.  `config.php` файлида база созламаларини (host, user, pass) текширинг.
4.  Тизимга кириш:
    - **Логин**: `admin`
    - **Пароль**: `12345` ёки `1998` (Админ томонидан ўзгартирилиши мумкин).
