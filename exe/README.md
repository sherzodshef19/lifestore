# 🖥️ LifeStore Desktop Launcher

Ушбу папка LifeStore веб-иловасини худди стандарт Windows дастури каби ишлаш имконини беради.

## 🚀 Биринчи марта ўрнатиш (Бошқа ПК-лар учун)

Агар сиз дастурни бошқа компьютерларга ўрнатмоқчи бўлсангиз, қуйидаги қадамларни бажаринг:

1.  Бутун `exe/` папкасини янги компьютерга нусхалаб олинг.
2.  `config.ini` файлини очинг ва `AppURL` манзилини онлайн сервердаги сайтингизга ўзгартиринг (масалан: `AppURL=https://sizning-saytingiz.uz`).
3.  `setup.bat` файлини икки марта босиб юргизинг.
4.  Сизнинг **Иш столингиз (Desktop)**-да `LifeStore POS` номли иконка пайдо бўлади.

## ⚙️ Созламалар (config.ini)

*   **AppURL**: Дастур уланиши керак бўлган веб-сайт манзили.
*   **AppName**: Дастур номи.

## 🛠️ Файлларга изоҳ:

*   **LifeStore.vbs**: Асосий "двигатель". У консолсиз, профессионал "App Mode"-да браузерни очади.
*   **setup.bat**: Иш столига (Desktop) дастур учун Shortcut (ёрлиқ) яратиб берадиган скрипт.
